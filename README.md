# Avianis

Avianis is a modern aviation management platform designed for private aviation and charter operations. This application provides a comprehensive suite of tools for flight scheduling, trip quoting, fleet management, and operational oversight.

## Features

-   **Trip Quoting**: Advanced calculator for estimating flight times, distances (NM), and generating quotes with support for different aircraft, accounts, and contract types.
-   **Schedule Management**: Tools for managing flight schedules and timelines.
-   **Operations**: Modules for managing airports, maintenance logs, and crew assignments.
-   **Fleet Management**: Tracking for aircraft and vendor details.
-   **Admin**: User management, finance monitoring, and company settings.

## Technology Stack

This project is built with a modern frontend stack ensuring performance and developer experience:

-   **Framework**: [React 19](https://react.dev/)
-   **Build Tool**: [Vite](https://vitejs.dev/)
-   **Language**: [TypeScript](https://www.typescriptlang.org/)
-   **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
-   **UI Components**: [Shadcn/UI](https://ui.shadcn.com/) (based on Radix UI)
-   **Routing**: [React Router v7](https://reactrouter.com/)
-   **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

### Prerequisites

-   Node.js (Latest LTS recommended)
-   npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    cd avianis-web
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:5173](http://localhost:5173) in your browser.

## Scripts

-   `npm run dev`: Start the development server.
-   `npm run build`: Build the application for production.
-   `npm run lint`: Run ESLint to check for code quality issues.
-   `npm run preview`: Preview the production build locally.
